# 1 helix配置
通过bash配置文件(~/.bashrc)配置vim为默认编辑器
	`export EDITOR=helix`
	`export VISUAL=helix`

创建helix的用户配置文件
	`$ touch ~/.config/helix/config.toml`
	`$ touch ~/.config/helix/languages.toml`

基础配置（进入config.toml文件）
	1. 主题选择
		`theme = "autumn"`
	2. 鼠标形状配置
		`[editor.cursor-shape]
		        insert = "bar"
			normal = "block"
			select = "underline"`
	3. 文件选择器配置
		`[editor.file-picker]
			hidden = false`
	4. 状态栏配置
		`[editor.statusline]
			left = ["mode", "spinner"]
			center = ["file-name"]
			right = ["diagnostics", "selections", "position", "file-encoding", "file-line-ending", "file-type"]
			separator = "│"
			mode.normal = "NORMAL"
			mode.insert = "INSERT"
			mode.select = "SELECT"
			diagnostics = ["warning", "error"]
			workspace-diagnostics = ["warning", "error"]`
	5. 按键映射配置
		`[keys.normal]
			C-s = ":w"
			C-x = ":q!"
			C-q = ":wq"
			C-down = "half_page_down"
			C-up = "half_page_up"`

lsp语言配置（进入language.toml文件）
	1. Bash语言配置
		`[[language]]
			name = "bash"
			indent = { tab-width = 8, unit = "\t" }`

# 2 helix使用
三种主要模式
	1. normal模式： 用于文件的导航和使用命令行，使用"Esc"键从insert,select模式进入
	2. insert模式： 用于文件的编辑，使用“i”键从normal,insert模式进入
	3. select模式： 用于文件的选择，使用“v”键从normal模式进入

导航功能
	1. 移动光标的方式
		- 按键
		- 鼠标
	2. 移动当前视图的方式
		- 使用"z"键展示功能列表
		- 使用绑定的快捷键
	3. 跳转到文件其他部分或其他文件
		- 使用“g”键展示功能列表
	4. 搜索与移动
		- 使用“/”键并输入搜索内容进行文件内搜索
		- 先选中字符，再使用“*”键进行文件内搜索
		- 在获取搜索结果后，使用“n”跳转到下一个匹配项，“N”跳转到上一个匹配项

基础功能
	1. 文件选择
		- f: 从当前文件LSP生效的根目录的文件中选择
		- F: 从当前文件所在目录的文件中选择
		- b: 从缓存区的文件中选择
		- g: 从当前文件所在git仓库的根目录的被修改文件中选择
