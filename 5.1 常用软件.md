# 2 stow
下载stow软件
	`sudo pacman -S stow`

使用**stow**软件,通过管理符号链接来管理软件包或用户配置文件,
将软件包或文件集中存储一个目录,通过符号链接给目标目录,便于集中管理分散的软件或配置文件
	- "stow directory": 存放软件和配置文件的根目录
	- "stow packagename directory": 不同软件和配置文件在"stow directory"下创建的各自子目录
	- "target directory": 存放符号链接的目录,默认是"stow directory"的父目录

使用**stow**命令创建符号链接时,保持"stow packagename direcory"下的层级结构,把根目录改为"target dirctory"
	`stow packagename`: 在默认目标目录处创建符号链接
	`stow -t target directory packagename`: 在指定的目标目录处创建符号链接

使用**stow**命令删除符号链接
	`stow -D packagename`

使用**stow**命令重新链接(先删除再创建,用于更新)
	`stow -R packagename`



# 3 fcitx5
安装相关软件:  
	`$ sudo pacman -S fcitx5 fcitx5-im fcitx5-chinese-addons`  

输入法配置:  
	1. 进入GUI配置界面  
  		`$ fcitx5-configtool`  
	2. 添加中文输入法: 进入“Input config”页面,从右边栏搜索“pingyin”并添加到左侧.
	3. 设置语言切换按键: 进入“Global Setting”页面,"Trigger Input Method"选择"Right Shift".
	4. 禁用临时语言切换案件: 进入“Global Setting”页面,"Temporally switch between first and current Input Method"选择为"empty".
	5. 设置默认主题: 进入“Addons”页面,进入"Classic User Interface"设置"Theme"为"DefaultDark"；设置字体为“noto sans”



# 4 vim
通过bash配置文件(~/.bashrc)配置vim为默认编辑器
	`export EDITOR=vim`
	`export VISUAL=vim`

创建vim的用户配置文件
	`$ vim ~/.vimrc`

基础功能配置
	1. 显示语法高亮
  		`syntax on`
	2. 换行自动缩进
  		`set autoindent`
	3. 设置搜索自动区分大小写，并实时高亮显示匹配项
  		`set ignorecase`
  		`set smartcase`
  		`set incsearch` 
	4. 启动鼠标支持
  		`set mouse=a`

按键配置
	1. 改变绑定的主键为空格
  		`let mapleader = " "`
	2. 自定义快捷键
  		`nnoremap <leader>w :w<CR>` 
  		`nnoremap <leader>q :q<CR>`
  		`nnoremap <leader>x :x<CR>`
  		`nnoremap <leader><Up> <C-u>`
  		`nnoremap <leader><Down> <C-d>`

配置从vim中复制内容到vim外的功能
	1. 下载相关软件
  		‘$ sudo pacman -S wl-clipboard’
	2. 选中要复制的内容，打开命令行，在已有内容后输入命令
  		‘w !wl-copy’	
	3. 在(~/.vimrc)配置命令的别名 
  		`cabbrev copy w !wl-copy`

>当需要通过系统权限使用编辑器编辑文件时，使用**sudoedit**命令,同时使用当前用户的vim配置
	`$ sudoedit filename`

# 7 firefox
下载firefox软件
	`$ sudo pacman -S firefox`

通过(~/.bashrc)设置别名，让其从终端启动后脱离终端运行
	`alias firefox='firefox & disown'`

更改浏览器外观设置
	1. 下载简体中文显示包
	2. 下载简体中文翻译包	
	4. 更改字体为“Noto Sans”
	3. 主页移除快捷方式栏
	5. 更改主题外观为“Dark”

更改搜索设置
	1. 默认使用“必应”搜索引擎，移除"百度"搜索引擎

更改浏览器隐私设置
	1. 选择"严格"的保护措施
	2. 要求网站不许出售和共享我的数据
	3. 不再询问保存我的密码

加载可能有的网页标签



# 8 steam  
# 8.1 下载steam
下载32位硬件驱动:
	1. 启动multilib库,编辑"/etc/pacman.conf"文件,取消[multilib]部分的注释
  		`$ sudoedit /etc/pacman.conf`
	2. 更新当前库  
  		`$ sudo pacman -Syy`  
	3. 安装32位OpenGL和Vulkan的AMD和Nvidia驱动  
  		`$ sudo pacman -S lib32-mesa lib32-vulkan-radeon lib32-vulkan-nouveau`

下载steam软件
	`sudo pacman -S steam`

通过(~/.bashrc)设置别名，让其从终端启动后脱离终端运行
	`alias steam='steam & disown`
 

# 8.2 使用兼容软件打开游戏 
在steam设置里找到兼容性，选择Proton的最新版本
右击游戏选择属性，找到兼容性，选择强制使用Proton打开游戏

> 可使用www.protondb.com查看游戏的兼容情况   
> 游戏默认放置路径
> 	- ~/.local/share/Steam/steamapps/common/<gamename>
> 	- ~/.local/share/Steam/steamapps/compatdata/<gamenID>/pfx/drive_c/
